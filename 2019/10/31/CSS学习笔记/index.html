CSS学习笔记

层叠样式表 (Cascading Style Sheets)

CSS 语法

主要的部分

选择器（selector），以及一条或多条声明 （declaration）（ 一个属性 和一个值组成 ）

属性（property）是您希望设置的样式属性（style attribute）

被分组的选择器就可以分享相同的声明

注意 :

  • 当使用 RGB 百分比时，即使当值为 0 时也要写 % 。

    但是在其他的情况下就不需要这么做了。尺寸像素0 之后不需要使用 px 单位。

  • 值为若干单词，则要给值加引号

  • 最后一条规则上是不需要加分号的

    但加分号的好处是，当你从现有的规则中增减声明时，会尽可能地减少出错的可能性

  • 空格、 大小写 不会影响 CSS 在浏览器的工作效果

    如果涉及到与 HTML 文档一起工作的话，class 和 id 名称对大小写是敏感的。

继承及其问题

根据 CSS，子元素从父元素继承属性

处理旧式 4.0 浏览器无法理解继承的问题 : “Be Kind to Netscape 4” 的冗余法则 ( 他们可以理解组选择器 )

摆脱父元素的规则: 创建一个针对 该子元素 的特殊规则

派生选择器

通过依据元素在其位置的上下文关系来定义样式
